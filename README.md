# 🐾 Pet Memories Gallery

A beautiful tribute to beloved pets. This gallery preserves cherished moments and memories with a soothing baby blue design.

## View the Gallery

Visit the live gallery at: **https://degenpotato.github.io/Potato/**

## Features

- 📸 Responsive photo gallery with smooth animations
- 🎬 Video support for special moments
- 💙 Gentle baby blue color scheme
- 📱 Mobile-friendly design
- 🖱️ Click to enlarge photos/videos
- ✨ Hover effects and smooth transitions

## How to Use

Simply open `index.html` in your browser or visit the GitHub Pages link above to view the gallery.
